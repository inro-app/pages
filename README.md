# inro pages

A shared collection of static HTML pages — slide decks, presentations, and other Claude-generated standalone pages — published via GitHub Pages.

**Live site:** https://inro-app.github.io/pages/

---

## Viewing pages

Each HTML file in this repo is available at:

```
https://inro-app.github.io/pages/<filename>.html
```

For example, `shopify_deck.html` is live at:
https://inro-app.github.io/pages/shopify_deck.html

---

## Adding a new page (with Claude Code)

You don't need to be technical to add a page. Follow these steps:

1. **Open Claude Code** in this repository
2. **Describe what you want** — e.g. *"Create a slide deck for our Q3 investor update"* or *"Turn these bullet points into a presentation"*
3. Claude will generate an HTML file and place it in the repo root
4. **Commit and push** — Claude can do this for you too, just ask

That's it. The page will be live at `https://inro-app.github.io/pages/<your-filename>.html` within a minute or two.

---

## File naming

- Use lowercase letters and hyphens: `q3-investor-deck.html`, `product-overview.html`
- No spaces, no capital letters
- Always `.html` extension
- Put files in the **root of the repo** (not in any subfolder)
